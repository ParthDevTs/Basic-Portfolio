FoodAdda Application

Site Description:

Food Adda Website is a Single Page Web Application developed using Angular 10.2 frontend and Java-SpringBoot Backend. Its a Restaurant booking website in which a restaurant vendor can display their Restaurants, Food Items, Prices and Booking Functionality and Customer can Search and book restaurants.
This is a site which can change functionality according to role selected without navigating.
